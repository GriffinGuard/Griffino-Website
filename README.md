***

This repository contains the source code, content, and assets for the Griffino website at [Griffino.cc](https://griffino.cc). It is a static Astro site published through GitHub Actions to GitHub Pages.

The Griffino — the open standard and local runtime for capability-based plugin composition: [Griffino Repo](https://github.com/GriffinGuard/Griffino). 

This repo only ships the marketing, documentation entry points, and repository index for griffino.cc.
